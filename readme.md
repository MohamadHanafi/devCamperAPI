# DevCamper API

> Backend API for DevCamper application, which is a bootcamp directory website.

## usage

Rename "config/config.env.env" to "config.env" and update the values to your own

## Install Dependencies

```
npm install
```

## Run app

```
# Run in dev mode
npm run dev

# Run in production mode
npm start
```
